# Webpack.config.js configuration

- Multi-Entry Points
- File Output Configuration
- Path Aliases
- Asset Import Handling
- CSS Support
- Module Optimization
- HTML Template Generation
- Automatic Clean Up
- Additional File Loaders for XML and CSV
- Development Mode
- Install Node modules:

Install Node modules:
```
npm install
```

Run
```
 npx webpack --config webpack.config.js
```
